# Student Grade Tracker

A Java-based console application to manage and analyze student marks.

## Project Description
This project is developed using Java as part of my CodeAlpha Java Programming Internship Task.

The Student Grade Tracker is a console-based application that takes student marks as input and calculates the average score, highest score, and lowest score.

## Technologies Used
- Java
- Scanner Class
- ArrayList

## Features
- Accepts student marks as input
- Stores marks using ArrayList
- Calculates average score
- Finds highest mark
- Finds lowest mark
- Displays summary report

## How to Run

Compile the program:

```bash
javac StudentGrade.java
```

Run the program:

```bash
java StudentGrade
```

## Output
The program displays student grade summary details.


# Hotel Reservation System

A Java-based console application to search, book, and manage hotel room reservations.

## Project Description
This project is developed using Java as part of my CodeAlpha Java Programming Internship Task.

The Hotel Reservation System allows users to search available rooms, make reservations, cancel bookings, and view booking details. The system manages different room categories and simulates the payment process.

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- File I/O
- Scanner Class

## Features
- Search available hotel rooms
- Room categorization (Standard, Deluxe, Suite)
- Make room reservations
- Cancel reservations
- Payment simulation
- View booking details
- Store booking information using File I/O

## Concepts Used
- Classes and Objects
- Constructors
- ArrayList
- Methods
- File Handling
- Exception Handling

## How to Run

Compile the program:

```bash
javac HotelReservation.java
```

Run the program:

```bash
java HotelReservation
```

## Output
The program displays a hotel management menu with options to search rooms, make bookings, cancel reservations, and view booking details.





# Stock Trading Platform

A Java-based console application to simulate a basic stock trading environment and manage user portfolio.

## Project Description

This project is developed using Java as part of my CodeAlpha Java Programming Internship Task.

The Stock Trading Platform allows users to view market data, buy and sell stocks, and track their portfolio performance. The system uses Object-Oriented Programming concepts to manage stocks, users, and transactions.

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList / HashMap
- Scanner Class
- File I/O

## Features

- Display available stock market data
- Buy stocks
- Sell stocks
- Manage user portfolio
- Track portfolio performance
- Calculate total portfolio value
- Store portfolio data using File I/O

## Concepts Used

- Classes and Objects
- Constructors
- Methods
- HashMap
- Collections
- File Handling
- Exception Handling

## How to Run

Compile the program:

```bash
javac StockTradingPlatform.java
```

Run the program:

```bash
java StockTradingPlatform
```

## Output

The program displays a stock trading menu where users can view market data, perform buy/sell operations, and track their portfolio value.
